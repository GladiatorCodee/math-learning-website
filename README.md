## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Select a grade and start learning!

## Deployment

### Netlify (Recommended)
1. Push to GitHub
2. Connect repository to Netlify
3. Deploy automatically (already configured via netlify.toml)

### GitHub Pages
1. Push to GitHub
2. Go to Settings > Pages
3. Select branch and deploy
